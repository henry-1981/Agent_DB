# test-legal Domain ŌĆö Rule ID Convention

## Format

```
tl-{source}-sec{N}-{suffix}
```

## Components

| Part | Description | Example |
|------|-------------|---------|
| `tl` | Domain prefix | `tl` |
| `{source}` | Source document short ID | `compliance` |
| `sec{N}` | Section number | `sec1` |
| `{suffix}` | `main` for main text, `sub{N}` for sub-clauses | `main`, `sub1` |

## Examples

```
tl-compliance-sec1-main     # Compliance Manual Section 1 main text
tl-compliance-sec2-sub1     # Section 2 sub-clause 1
```
